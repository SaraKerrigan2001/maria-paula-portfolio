# Hoja de Vida - María Paula Capacho González

## 🚀 Diseño Moderno Inspirado en Jeric Pulla

Este proyecto es una hoja de vida moderna desarrollada con Next.js 16 y Tailwind CSS, inspirada en el diseño futurista de Jeric Pulla con colores teal/turquesa y efectos de vidrio.

### ✨ Características

- **Diseño Futurista**: Inspirado en portfolios modernos con efectos de vidrio y neón
- **Selector de Idioma**: Cambio entre Español, Inglés y Francés con banderas
- **Chatbot Inteligente**: Asistente virtual interactivo con respuestas contextuales
- **Responsive**: Adaptable a todos los dispositivos
- **Animaciones Suaves**: Transiciones y efectos hover elegantes
- **Colores Teal/Turquesa**: Paleta de colores moderna y atractiva
- **Tipografía Bold**: Textos impactantes con efectos de gradiente

### 🛠️ Tecnologías Utilizadas

- **Next.js 16**: Framework de React para aplicaciones web
- **Tailwind CSS 4**: Framework de CSS utilitario
- **TypeScript**: Tipado estático para JavaScript
- **React 19**: Biblioteca de JavaScript para interfaces de usuario

### 🎨 Secciones

1. **Hero/Landing**: Sección principal con login y presentación
2. **About**: Información personal con redes sociales
3. **Portfolio**: Galería de proyectos con efectos visuales
4. **Contact**: Formulario de contacto
5. **Selector de Idioma**: Cambio entre ES, EN, FR en el navbar
6. **Chatbot**: Asistente virtual inteligente con respuestas contextuales

### 🚀 Instalación y Uso

```bash
# Clonar el repositorio
cd hoja-vida-nextjs

# Instalar dependencias
npm install

# Ejecutar en modo desarrollo
npm run dev

# Construir para producción
npm run build

# Ejecutar en producción
npm start
```

### 📝 Personalización

Para personalizar el contenido:

1. **Información Personal**: Editar `src/components/About.tsx`
2. **Proyectos**: Modificar el array `projects` en `src/components/Portfolio.tsx`
3. **Imágenes**: Reemplazar archivos en `public/img/`
4. **Colores**: Ajustar variables CSS en `src/app/globals.css`
